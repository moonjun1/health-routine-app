# 🏋️ GymRoutine 앱 사용 가이드

## 📱 앱 개요
GymRoutine은 헬스장 보유 기구를 기반으로 AI가 맞춤 운동 루틴을 생성해주는 앱입니다.

---

## 🚀 시작 가이드 (처음 사용자)

### 1단계: 회원가입 / 로그인 ✅
```
홈 화면 → 프로필 탭 → "로그인" 버튼 클릭
→ 이메일/비밀번호로 회원가입 또는 로그인
```

**💡 팁**: 로그인하지 않아도 헬스장 정보는 볼 수 있지만, 루틴 생성과 저장은 로그인이 필요해요.

---

### 2단계: 헬스장 등록 🏢 (필수!)
```
홈 화면 → "헬스장 검색" 버튼 클릭
→ 주소나 헬스장 이름으로 검색
→ 원하는 헬스장 선택
→ 보유 기구 선택 (최소 1개 이상 필수!)
→ "등록" 버튼 클릭
```

**⚠️ 중요**:
- 헬스장의 보유 기구를 반드시 선택해야 해요
- 기구 목록이 없으면 AI 루틴 생성이 안 돼요
- 여러 헬스장을 등록할 수 있어요

**📌 기구 선택 예시**:
- ✅ 바벨, 덤벨, 벤치프레스, 스쿼트랙, 레그프레스...
- 최소 5~10개 이상 선택하는 것을 추천해요

---

### 3단계: AI 루틴 생성 🤖
```
홈 화면 → "AI 루틴" 버튼 클릭
또는 하단 "루틴" 탭 → "+" 버튼 → "AI 루틴 생성"
```

#### AI 루틴 생성 화면 입력 항목:

**1) 헬스장 선택** 🏢
```
드롭다운에서 등록한 헬스장 중 선택
→ 선택한 헬스장의 보유 기구만 사용해서 루틴 생성
```

**2) 운동 목표** 🎯
- 근력 증가
- 체중 감량
- 체력 향상
- 근비대
- 지구력

**3) 운동 경력** 💪
- 초보자: 6개월 미만
- 중급자: 6개월 ~ 2년
- 고급자: 2년 이상

**4) 주당 운동 횟수** 📅
- 슬라이더로 3~6회 선택

**5) 운동 시간** ⏱️
- 슬라이더로 30~120분 선택

**6) 선호하는 운동 부위** 🎯
- 가슴, 등, 어깨, 팔, 하체, 코어 중 선택 (다중 선택 가능)

**7) 추가 정보 (선택사항)** 📝
```
예시:
- "허리 부상으로 데드리프트 제외"
- "무릎이 안 좋아서 스쿼트 대신 레그프레스 선호"
- "가슴 운동 비중을 높이고 싶어요"
```

**8) "AI 루틴 생성" 버튼 클릭** 🚀

---

### 4단계: 생성된 루틴 확인 및 저장 💾
```
AI가 루틴 생성 완료 → 팝업에서 운동 목록 확인
→ "루틴 저장" 버튼 클릭
→ "내 루틴" 목록에 추가됨
```

**생성된 루틴 정보**:
- 루틴 이름
- 운동 카테고리
- 운동 목록 (세트/횟수/휴식시간 포함)
- 각 운동에 대한 설명

---

### 5단계: 저장된 루틴 사용 🏃‍♂️
```
하단 "루틴" 탭 → 저장된 루틴 선택
→ 운동 시작
→ 각 세트 완료 시 체크
→ 운동 완료 후 기록 저장
```

---

## 🔄 일반적인 사용 흐름

### 매일 운동할 때:
```
1. 앱 실행
2. "루틴" 탭 → 오늘 할 루틴 선택
3. 운동 시작
4. 세트별로 완료 체크
5. 운동 완료
```

### 새로운 루틴이 필요할 때:
```
1. "루틴" 탭 → "+" 버튼
2. "AI 루틴 생성" 선택
3. 조건 입력 (목표, 경력, 횟수 등)
4. 루틴 생성 및 저장
```

### 다른 헬스장 갈 때:
```
1. 홈 화면 → "헬스장 검색"
2. 새 헬스장 등록
3. AI 루틴 생성 시 드롭다운에서 새 헬스장 선택
4. 해당 헬스장 기구로 루틴 생성
```

---

## ⚠️ 자주 하는 실수

### 1. AI 루틴 생성 버튼이 비활성화되어 있어요
**원인**:
- ❌ 헬스장을 선택하지 않음
- ❌ 운동 목표를 선택하지 않음
- ❌ 선호 운동 부위를 선택하지 않음

**해결**:
- ✅ 위 3가지를 모두 입력해야 버튼이 활성화됩니다

### 2. "선택한 헬스장에 등록된 기구가 없습니다" 오류
**원인**:
- ❌ 헬스장 등록 시 보유 기구를 선택하지 않음

**해결**:
```
홈 화면 → 내 헬스장 카드 → "변경" 버튼
→ 헬스장 선택 → 보유 기구 선택
→ "등록" 버튼
```

### 3. 헬스장 정보가 안 보여요
**원인**:
- ❌ 헬스장을 아직 등록하지 않음

**해결**:
- ✅ 홈 화면에서 "헬스장 검색" → 헬스장 등록

## 📋 기능별 상세 가이드

### 헬스장 관리

#### 헬스장 등록하기
```
홈 → "헬스장 검색" → 검색 → 선택 → 기구 선택 → 등록
```

#### 헬스장 변경하기
```
홈 → 헬스장 카드 → "변경" 버튼
```

#### 여러 헬스장 등록하기
```
1. 첫 번째 헬스장 등록
2. 다시 "헬스장 검색" → 두 번째 헬스장 등록
3. AI 루틴 생성 시 드롭다운에서 선택 가능
```

**💡 활용 예시**:
- 집 근처 헬스장
- 회사 근처 헬스장
- 출장지 헬스장
→ 각각 등록해두고 상황에 맞게 선택!

---

### 루틴 관리

#### 루틴 목록 보기
```
하단 "루틴" 탭
```

#### 루틴 수정하기
```
루틴 탭 → 루틴 선택 → "편집" 버튼
```

#### 루틴 삭제하기
```
루틴 탭 → 루틴 선택 → "삭제" 버튼
```

---

### 운동 기록

#### 운동 시작하기
```
루틴 선택 → "운동 시작" 버튼
```

#### 세트 완료 체크
```
각 운동의 세트별로 체크박스 클릭
```

#### 운동 기록 보기
```
프로필 탭 → "운동 기록" (또는 홈 화면에 최근 운동 표시)
```

## 💡 효과적인 사용 팁

### 1. 헬스장 기구 정보를 정확하게 입력하세요
```
✅ 좋은 예:
- 바벨, 덤벨, 벤치프레스, 스쿼트랙, 레그프레스, 렛풀다운,
  케이블 크로스오버, 레그 익스텐션, 레그 컬, 숄더프레스,
  체스트 프레스 머신 등 (10개 이상)

❌ 나쁜 예:
- 바벨, 덤벨 (2개만)
→ 생성되는 루틴의 운동 종류가 너무 제한적입니다
```

### 2. 목표에 맞게 조건을 설정하세요
```
근육량 증가 목표:
→ 근비대 / 중급자 / 4~5회/주 / 60~90분 / 모든 부위 선택

체중 감량 목표:
→ 체중 감량 / 초보자 / 3~4회/주 / 45~60분 / 하체+코어 중심

체력 향상 목표:
→ 체력 향상 / 초보자 / 3~4회/주 / 30~45분 / 전신 운동
```

### 3. 추가 정보를 활용하세요
```
예시:
- "상체 집중 루틴 원해요"
- "프리웨이트 중심으로 구성해주세요"
- "어깨 부상 회복 중이라 가벼운 중량으로"
- "대회 준비중이라 고강도로"
```

### 4. 여러 루틴을 만들어두세요
```
- 상체 집중 루틴 (월, 목)
- 하체 집중 루틴 (화, 금)
- 전신 루틴 (수)
→ 요일별로 다른 루틴 활용
```

## 🆘 문제 해결 (Troubleshooting)

### Q: AI 루틴 생성이 느려요
**A**:
- 첫 생성은 10~30초 정도 걸릴 수 있어요
- 인터넷 연결 상태를 확인해주세요

### Q: 생성된 루틴이 마음에 안 들어요
**A**:
- 조건을 바꿔서 다시 생성해보세요
- "추가 정보"에 더 구체적인 요구사항을 입력해보세요
- 원하는 부위를 "선호 운동 부위"에 선택했는지 확인하세요

### Q: 로그인이 안 돼요
**A**:
- 이메일/비밀번호를 확인해주세요
- 회원가입을 먼저 진행했는지 확인해주세요
- 인터넷 연결 상태를 확인해주세요

### Q: 헬스장을 여러 개 등록했는데 어떻게 선택하나요?
**A**:
- AI 루틴 생성 화면 상단에 "헬스장 선택" 드롭다운이 있어요
- 그곳에서 원하는 헬스장을 선택하세요
